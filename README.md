# Med-SEAL Suite  - Documentation

[![Documentation Status](https://readthedocs.org/projects/med-seal/badge/?version=latest)](https://med-seal.readthedocs.io/)

Comprehensive documentation for the **Med-SEAL Suite**  - an open-source healthcare platform built on international standards.

## Build Locally

```bash
cd docs
pip install -r requirements.txt
make html
open _build/html/index.html
```

## Structure

| Section | Description |
|---|---|
| **Overview** | Architecture, getting started |
| **Components** | OpenEMR, Medplum, Patient Portal Native, SSO |
| **AI Platform** | 6 agents, 18 features, SEA-LION Guard |
| **Operations** | Deployment, scripts, standards, FHIR resources |
